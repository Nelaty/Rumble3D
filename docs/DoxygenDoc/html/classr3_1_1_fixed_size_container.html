Detailed Description

template<class Element_Type, class Container_Type = std::vector<Element_Type>>
class r3::FixedSizeContainer< Element_Type, Container_Type >

Template class for a container, which can be resized, with random access (dependent on Container_Type) and can be filled similar to a stack. Its size in memory will stay the same at all times. Elements can not be erased from it.

Template Parameters
Element_TypeType of the elements, saved in this container.
Container_TypeContainer type used for storing elements. Needs to have a "void resize(unsigned)" function and must be able to store elements of type Element_Type
